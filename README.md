# 6-Layer PCB-Based BLDC Motor Stator Design

## 📌 Overview
This repository presents the design of a 6-layer planar PCB stator for a Brushless DC (BLDC) motor. The stator windings are implemented using multi-layer copper traces arranged in a radial geometry to form balanced three-phase windings.

The multi-layer stack enables increased ampere-turn density, improved current handling capability, and compact axial motor construction.

---

## ⚙️ Key Features
- 6-Layer PCB Stackup
- Radial copper winding topology
- Multi-layer interconnected phase windings
- Symmetrical three-phase distribution
- High-current trace design
- Central rotor clearance
- Precision mounting hole alignment

---

## 🛠 Design Highlights
- Series-connected winding segments across layers
- Optimized copper width for current density control
- Electromagnetic symmetry for smooth torque generation
- Compact planar architecture suitable for axial-flux BLDC motors
- Balanced mechanical layout to reduce vibration

---

## 🧲 Architecture
- Planar multi-layer stator
- Copper windings distributed across 6 layers
- Interlayer vias used for vertical current transfer
- Designed for compact axial BLDC motor applications
